◆ traversed

bool Engine.Physics.BEPUik.Bone.traversed
package

Used by the per-control traversals to find stressed paths. It has to be separate from the IsActive flag because the IsActive flag is used in the same traversal to denote all visited bones (including unstressed ones). Also used in the unstressed traversals; FindCycles uses the IsActive flag and the following DistributeMass phase uses the traversed flag.

◆ unstressedCycle
