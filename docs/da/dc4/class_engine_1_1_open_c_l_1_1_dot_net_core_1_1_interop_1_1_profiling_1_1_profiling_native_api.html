Returns profiling information for the command associated with event if profiling is enabled.

Parameters
eventHandleSpecifies the OpenCL event being queried.
parameterNameAn enumeration constant that specifies the information to query.
parameterValueSizeA pointer to memory where the appropriate result being queried is returned. If parameterValue is null, it is ignored.
parameterValueSpecifies the size in bytes of memory pointed to by parameterValue.
parameterValueSizeReturnedReturns the actual size in bytes of data being queried by parameterValue. If parameterValueSizeReturned is null, it is ignored.
Returns

Returns Result.Success if the function executed successfully, or one of the errors below:

Result.ProfilingInformationNotAvailable if the CommandQueueProperty.ProfilingEnable flag is not set for the command-queue, if the execution status of the command identified by event is not complete or if event refers to the EnqueueSvmFree command or is a user event object.

Result.InvalidEvent if eventHandle is not a valid event.

Result.InvalidValue if parameterName is not one of the supported values or if size in bytes specified by parameterValueSize is less than the size of the return type parameterValue is not a null value.

Result.OutOfResources if there is a failure to allocate resources required by the OpenCL implementation on the device.

Result.OutOfHostMemory if there is a failure to allocate resources required by the OpenCL implementation on the host.
