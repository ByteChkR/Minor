◆ blendAmount

float Engine.Physics.BEPUphysics.EntityStateManagement.InterpolatedStatesManager.blendAmount
private

summary> Gets or sets the blending amount to use. This is set automatically when the space is using internal timestepping (I.E. when Space.Update(dt) is called). It is a value from 0 to 1 that defines the amount of the previous and current frames to include in the blended state. A value of 1 means use only the current frame; a value of 0 means use only the previous frame. /summary>
