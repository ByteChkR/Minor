Detailed Description

Represents an event, which is returned by all OpenCL methods, that take longer. They can be used await asynchronous API calls. This class is awaitable and can be used with the C# await keyword. Please not, that when the awaitable event is awaited, then it is auto-disposed, but when the awaitable event is not awaited, then it must be disposed of manually.
