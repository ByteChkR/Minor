void Engine.Physics.BEPUphysics.BroadPhaseEntries.MobileCollidables.EntityCollidable.UpdateBoundingBoxForTransform ( ref RigidTransform  transform,
float  dt 
)

Updates the collidable's world transform and bounding box. The transform provided will be offset by the collidable's LocalPosition to get the shape transform. This is a convenience method for external modification of the collidable's data.

Parameters
transformTransform to use for the collidable.
dtDuration of the simulation time step. Used to expand the bounding box using the owning entity's velocity. If the collidable does not have an owning entity, this must be zero.
