void Engine.Physics.BEPUphysics.Constraints.JointBasis3D.SetLocalAxes ( Matrix3x3  matrix)

Sets up the axes of the transform and ensures that it is an orthonormal basis.

Parameters
matrixRotation matrix representing the three axes. The matrix's backward vector is used as the primary axis.
The matrix's right vector is used as the x axis. The matrix's up vector is used as the y axis.
