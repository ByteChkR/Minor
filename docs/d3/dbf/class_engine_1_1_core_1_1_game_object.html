void Engine.Core.GameObject.applyRenderHierarchy ( bool  hasRenderer)
private

Optimization. If a Renderer Was attached to this object or one object in the children we set the Has Renderer flag to true. The other way around if a Renderer gets removed(but we stop unsetting the flag when we found another child with a renderer attached) This way we know which matrixes we need to compute and which ones we can discard since they wont show up in the renderer anyway
