Detailed Description

Provides storage for reusable arrays with power-of-2 lengths.

Template Parameters
TType of resource contained in the buffers.

This is designed for use with unsafe code. It often sacrifices safety for performance or simplicity. Running with DEBUG defined will catch some misuse, but otherwise many invalid usages will be allowed.
