Detailed Description

Collidable mesh which can be created from a reusable InstancedMeshShape. Very little data is needed for each individual InstancedMesh object, allowing a complicated mesh to be repeated many times. Since the hierarchy used to accelerate collisions is purely local, it may be marginally slower than an individual StaticMesh. /summary>
